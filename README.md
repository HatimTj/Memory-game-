# 🃏 Memory Game

Un jeu de mémoire interactif développé avec **React**, **TypeScript** et **Vite**.

## 🎮 Description

Le joueur doit retrouver des paires de cartes identiques parmi un ensemble de cartes cachées, en un minimum de tentatives et dans le temps imparti.

## ✨ Fonctionnalités

- 🔀 Mélange aléatoire des cartes à chaque partie
- ⏱️ Chronomètre intégré
- 🔢 Compteur de tentatives
- 🏆 Système de meilleur score (localStorage)
- 📱 Interface responsive (PC, tablette, mobile)
- 🎨 Animations CSS fluides

## 🛠️ Technologies

- React
- TypeScript
- Vite
- Tailwind CSS
- ESLint
- PostCSS
- Lucide React

## 🚀 Installation
```bash
# Cloner le projet
git clone https://github.com/HatimTj/Memory-game-.git

# Installer les dépendances
npm install

# Lancer le serveur de développement
npm run dev
```
